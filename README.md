# Telco Customer Churn Analysis

## Project Overview

This project analyzes customer churn data from a telecommunications company to identify factors influencing customer attrition. The analysis was performed using Excel, SQL, Python, and Power BI to generate actionable business insights.

## Problem Statement

Customer churn is a major challenge for telecom companies because losing customers directly impacts revenue and growth. The objective of this project is to analyze customer behavior, identify churn patterns, and provide recommendations to improve customer retention.

## Tools Used

* Microsoft Excel
* SQL
* Python
* Power BI
* VS Code

## Project Workflow

1. Data Cleaning using Excel
2. Data Exploration using SQL
3. Exploratory Data Analysis (EDA) using Python
4. Dashboard Development using Power BI
5. Business Insights and Recommendations

## Dataset Information

* Dataset: Telco Customer Churn
* Total Customers: 7,046
* Churned Customers: 1,869
* Churn Rate: 26.53%

## Key Findings

* Customers with month-to-month contracts have the highest churn rate.
* Fiber optic customers are more likely to churn.
* Electronic check users show higher churn behavior.
* Long-term contract customers are less likely to leave.

## Power BI Dashboard

### KPI Metrics

* Total Customers
* Churned Customers
* Churn Rate
* Average Monthly Charges

### Visualizations

* Churn by Contract Type
* Churn by Internet Service
* Churn by Payment Method
* Customer Segmentation

## Business Recommendations

1. Encourage customers to move from month-to-month plans to annual contracts.
2. Improve retention strategies for fiber optic customers.
3. Investigate customer experience issues related to electronic check payments.
4. Offer loyalty programs to long-tenure customers.
